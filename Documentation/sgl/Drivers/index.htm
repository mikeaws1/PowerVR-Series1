PowerVR Series1 Driver Releases

You can download the latest PowerVR Series1 developer drivers from here. For your convenience, two packages are provided for each driver version (from 4.1.0c3 onwards) - the plain driver release, and another package including the new PowerVR Debug Utilities .

From version 4.1.1 onwards the application specific settings (AppHints) and default configuration settings of the OEM product drivers are tracked in the developer drivers.

This has been introduced to resemble the PowerVR product driver functionality and default settings in the development environment to streamline development and testing efforts.

However, we recommend that for QA testing purposes you use the appropriate OEM product drivers (e.g. Matrox m3D and VideoLogic Apocalypse 3D / 3Dx).
