Welcome to the PowerVR Series1 Developer Support WWW Area!

On these pages you will find all the information you need to develop 3D programs using PowerVR Series1 technology.

Here is a brief overview :

The QuickStart section will allow you to get a single triangle up and running in minutes. Some further steps are presented in a brief tutorial which will change this into an animated set of textured triangles immersed in thick blue fog. After some playing around with this you may want to know the full details...

Solutions

 Tips, Tricks and Code Fragments are provided to give you examples of some cool PowerVR features.

We also have a Frequently Asked Questions section.
In the Documentation section you will find electronic documents for developing with each API. You can choose between Microsoft Word and Adobe Acrobat formats.

Documentation

Drivers

 The Driver section allows you to download the Driver Package for the latest manufacturing release. You can also check out the latest developments by testdriving the latest beta drivers. These will be of interest if you would like to see the improvements that the driver team is making, such as bug fixes and performance enhancements. Recent archives are also available on this page.
You will find example code and applications in the Sample Programs section. This will be a good place to start as it will show you some of the features of PowerVR.

Samples

Technology

 If you are interested in PowerVR technology, you may find the Technology section interesting, which tells you all about the latest PowerVR hardware, and its advantages over conventional 3D hardware solutions.
If your PowerVR compatible title is currently at an advanced stage of development, you will most probably be planning launch activities and thinking about how to make the maximum market impact at launch and how to generate end user demand. You will then certainly find our Logo Certification Program [which is no longer available] exciting, which is designed to help you make a successful launch of your title.
