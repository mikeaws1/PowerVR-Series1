Direct3D Immediate Mode Table Fog

 

For fogging effects, we strongly recommend using table fog as opposed to vertex fog on PowerVR PCX1/PCX2. That is due to the fact that table fog is supported by the hardware and has no performance implications.
This example program shows how to do table fog in immediate mode Direct3D. Source and Executable are provided.
