Chroma Key

 

PowerVR PCX1 and PCX2 do not natively support colour key. However, colour key is commonly used as an easy way to store textures with transparency information.

PowerSGL Direct Example

This example program illustrates how colour key textures can be used on PowerVR PCX1/2 by creating an appropriate alpha texture map. This texture is created by setting the alpha value to fully transparent for those texels which have the chroma key colour. The RGB values are re-calculated in order to avoid bilinear interpolation of the chroma key colour into the texture map.

In this demo, you may use the cursor keys to alter the global translucency. This solution can also be applied in Direct3D applications using ARGB 4444 textures.
