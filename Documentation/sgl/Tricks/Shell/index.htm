SGL Direct / Direct3D Shell Environment

 

SGL Direct and Direct3D Shell

The PowerVR Shell provides a safe and flexible environment for your 3D application to run in.
It consists of a single .c and a single .h file.

The PowerVR Shell allows the user to change various settings including FullScreen/Window mode operation, buffering mode and screen resolution at run-time.

There are two versions of the PowerVR Shell, a D3D version (which should also work on hardware other than PowerVR) and an SGL Direct version:

SGL Direct Shell 32 KB
Direct3D Shell 45 KB
